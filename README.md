# HoloVan

**HoloVan** is a simple, short 2D game created for the **GIGA Gamejam 2025**. This project explores a unique blend of 2D mechanics and thematic storytelling, all developed within a limited jam timeframe.

## 📺 Gameplay Showcase
Check out the gameplay video here: [Watch on YouTube](https://youtu.be/Jv-sfXXLlcQ)

## 🛠️ Built With
* **Engine**: [Unity](https://unity.com/)
* **Language**: C#
* **Graphics API**: DirectX 12 / Vulkan support

## 🚀 How to Play
1.  **Download** this repository.
2.  **Extract** the contents to a folder on your computer.
3.  Run **`HoloVan.exe`**.
4.  Enjoy the game!
